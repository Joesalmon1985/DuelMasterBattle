# Product Designer Role

## Responsibility
Player-facing rules clarity and encounter progression framing.

## For this project
- Wizard-duel reskin of duelling Mastermind with Hit / Weakness / Unaffected feedback.
- **Encounter progression:** tutorial (Blue Apprentice) → full duel (Archmage); asymmetric secret vs attack pools and enemy tells per encounter.
- Alternating one-attack-per-turn flow with dual visible histories.
- Point names and slot count vary by encounter (1–4).
- Accessibility: magic types identified by label, number, and symbol — not colour alone.
