# Product Designer Role

## Responsibility
Clarify UX, board layout, game flow, win/loss/draw rules.

## For this project
- Two-board Mastermind-inspired layout adapted for sequential Human vs Bot.
- Human secret setup: 4 clickable peg slots → 10-colour picker → lock when all 4 set.
- Bot guess board: 12 rows × 4 pegs + feedback markers per row.
- Status text: phase (setup / bot guessing / human guessing / result), remaining guesses.
- Result screen: who solved what, guess counts, restart button.

## Acceptance
Flow matches the 14-step PRIMARY SUCCESS CONDITION in PLAN.md.
