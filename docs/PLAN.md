# Duel Master Battle — Project Plan

## Goal
Build a playable **Human vs Bot** duelling Mastermind vertical slice in Godot 4.4.x, backed by tested pure game rules in Python and GDScript with shared golden fixtures.

## Primary success condition
Launch Godot → Human vs Bot → set 4-peg secret code via clickable pegs + 10-colour chooser → lock code → bot makes up to 12 visible guesses with feedback → bot sets hidden code → human guesses bot code → result screen → restart.

## Tech stack
| Layer | Choice |
|-------|--------|
| Rules (Python) | `python_prototype/duel_mastermind/` + pytest |
| Rules (Godot) | `godot_project/sim/` + headless `run_tests.gd` |
| Python UI | tkinter (minimal proof only) |
| Godot UI | Godot 4.4.x 2D Control-based scenes |
| Parity | `shared_fixtures/` JSON golden files |
| Bot | Random legal bot (solver deferrable) |

## Architecture
```
shared_fixtures/     ← generated by Python, consumed by both
python_prototype/    ← pure logic + minimal tkinter
godot_project/
  sim/               ← authoritative rules (headless)
  client/            ← UI scenes (non-authoritative)
tools/               ← test runners, cleanup scripts
docs/                ← rules, milestones, playtest
```
